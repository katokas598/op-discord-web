# Инструкция по запуску DiscordSer Bot v2.0

## Быстрый старт

1. **Первый запуск**: Дважды кликните на `start_bot.bat`
   - Будет автоматически запущена настройка через setup.py
   - Введите токен Discord бота и другие настройки

2. **Запуск панели**: Дважды кликните на `start_dashboard.bat`
   - Web-панель будет доступна на http://localhost:8000

## Получение токена Discord бота

1. Идите на https://discord.com/developers/applications
2. Создайте новое приложение (New Application)
3. Перейдите в раздел "Bot" 
4. Нажмите "Add Bot"
5. Скопируйте токен из раздела "Token"

## Настройка прав бота

Бот нуждается в следующих правах:
- `Manage Messages` - для очистки сообщений
- `Kick Members` - для кика участников  
- `Ban Members` - для бана участников
- `Manage Channels` - для создания тикетов
- `Manage Permissions` - для мута участников
- `Read Message History` - для чтения истории
- `Send Messages` - для отправки сообщений
- `Use Slash Commands` - для команд

## Приглашение бота на сервер

1. В Discord Developer Portal перейдите в раздел "OAuth2" -> "URL Generator"
2. Выберите scope: `bot`
3. Выберите нужные permissions (права указаны выше)
4. Скопируйте сгенерированную ссылку и перейдите по ней
5. Выберите сервер и добавьте бота

## Команды бота

### Основные
- `!ping` - проверить бота
- `!cmds` - список команд
- `!serverinfo` - информация о сервере
- `!userinfo [@пользователь]` - информация о пользователе

### Модерация (только для администраторов)
- `!ban @пользователь [причина]` - забанить
- `!unban <id>` - разбанить по ID
- `!kick @пользователь [причина]` - кикнуть
- `!mute @пользователь [время] [причина]` - замутить (10m, 1h, 1d)
- `!unmute @пользователь` - размутить
- `!warn @пользователь [причина]` - выдать предупреждение
- `!warns [@пользователь]` - посмотреть предупреждения
- `!clearwarns @пользователь` - очистить предупреждения
- `!clear <количество>` - очистить сообщения (1-1000)
- `!lock` - заблокировать канал
- `!unlock` - разблокировать канал

### Тикеты
- `!ticket [категория]` - создать тикет
- `!closeticket` - закрыть тикет (только модераторы)

## Web-панель

После запуска dashboard будет доступна на http://localhost:8000

Функции панели:
- Просмотр участников сервера
- Модерация (бан, кик, мут, предупреждения)
- Управление тикетами
- Просмотр логов модерации
- Настройка пользовательских команд
- Статистика сервера

## Файлы проекта

- `main.py` - основной файл запуска бота
- `config.json` - конфигурация (создается автоматически)
- `setup.py` - мастер настройки
- `dashboard_server.py` - web-панель управления
- `bot.db` - база данных SQLite (создается автоматически)
- `discord_bot/bot.py` - логика Discord бота
- `database.py` - функции для работы с БД
- `index.html`, `styles.css`, `script.js` - файлы web-интерфейса

## Устранение неполадок

### Ошибка "Invalid Token"
- Проверьте правильность токена Discord бота
- Убедитесь что токен скопирован полностью
- Перезапустите setup.py для ввода нового токена

### Ошибка "Missing Permissions"  
- Проверьте права бота на сервере
- Убедитесь что роль бота выше ролей пользователей, которых модерируете

### Web-панель не открывается
- Проверьте что порт 8000 не занят другим приложением
- Убедитесь что запущен dashboard_server.py
- Попробуйте другой браузер

### Проблемы с кодировкой в консоли
- Это нормально для Windows, функциональность не нарушена
- Используйте .bat файлы для запуска

## Поддержка

При возникновении проблем:
1. Проверьте все настройки в config.json
2. Убедитесь что бот имеет необходимые права
3. Проверьте логи в консоли на наличие ошибок
