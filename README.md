# DiscordSer Bot

Мультифункциональный бот для Discord с красивой панелью управления и авто-setup.

## Что есть
- Discord-команды для модерации
- Тикеты
- Логи и варны
- Красивый сайт с анимациями и разделами управления
- Локальная админ-панель для настройки и модерации

## Запуск бота

```bash
pip install -r requirements.txt
python setup.py
python main.py
```

## Запуск панели

Запустите бота:

```bash
python main.py
```

В другом окне запустите панель:

```bash
python dashboard_server.py
```

Потом откройте `http://localhost:8000`

Панель работает вместе с живым ботом и выполняет реальные действия на сервере.

## Сайт
- Быстрая настройка бота
- Пользователи сервера
- Бан / кик / мут / warn
- Тикеты и жалобы
- Логи и статистика
- Кастомные команды
- Настройки бота
- Админ-панель
- Модерация и статистика
