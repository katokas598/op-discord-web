# DiscordSer Bot v2.0

Мультифункциональный бот для Discord с красивой панелью управления и автоматической настройкой.

## ✨ Возможности
- 🤖 Discord-команды для модерации (бан, кик, мут, предупреждения)
- 🎫 Система тикетов с категориями
- 📊 Логирование всех действий модерации
- 🌐 Web-панель управления с красивым интерфейсом
- ⚙️ Автоматическая настройка через мастер
- 🔧 Пользовательские команды
- 📈 Статистика сервера и участников

## 🚀 Быстрый запуск

### Вариант 1: Через .bat файлы (Windows)
1. Дважды кликните `start_bot.bat` - настройка и запуск бота
2. Дважды кликните `start_dashboard.bat` - запуск web-панели

### Вариант 2: Через командную строку
```bash
# Установка зависимостей
pip install -r requirements.txt

# Первоначальная настройка
python setup.py

# Запуск бота
python main.py

# В другом терминале: запуск панели управления
python dashboard_server.py --no-prompt
```

## 📋 Что нужно для работы

1. **Python 3.8+** 
2. **Discord Bot Token** - получите на https://discord.com/developers/applications
3. **ID вашего Discord сервера** - включите режим разработчика в Discord и скопируйте ID

## 🌐 Web-панель

После запуска панель доступна на `http://localhost:8000`

**Функции:**
- Просмотр и модерация участников
- Управление тикетами и жалобами  
- Просмотр логов модерации
- Настройка пользовательских команд
- Статистика сервера

## 🤖 Команды бота

### Основные
- `!ping` - проверить работу бота
- `!cmds` - показать список команд
- `!serverinfo` - информация о сервере
- `!userinfo [@user]` - информация о пользователе
- `!invite` - создать приглашение

### Модерация (для администраторов)
- `!ban @user [reason]` - забанить пользователя
- `!unban <id>` - разбанить по ID
- `!kick @user [reason]` - кикнуть пользователя  
- `!mute @user [time] [reason]` - замутить (10m, 1h, 1d)
- `!unmute @user` - размутить
- `!warn @user [reason]` - выдать предупреждение
- `!warns [@user]` - показать предупреждения
- `!clearwarns @user` - очистить предупреждения
- `!clear <amount>` - удалить сообщения (1-1000)
- `!lock` / `!unlock` - заблокировать/разблокировать канал

### Тикеты
- `!ticket [category]` - создать тикет
- `!closeticket` - закрыть тикет (модераторы)

## ⚙️ Настройка

Все настройки хранятся в `config.json`:

```json
{
  "discord_token": "ВАШ_ТОКЕН_БОТА",
  "prefix": "!",
  "guild_id": "ID_ВАШЕГО_СЕРВЕРА", 
  "admin_ids": [123456789],
  "max_warns": 3,
  "ticket_categories": ["Техподдержка", "Жалобы", "Предложения"],
  "dashboard_domain": "your-domain.com",
  "dashboard_host": "0.0.0.0",
  "dashboard_port": 8000
}
```

## 🔧 Структура проекта

- `main.py` - точка входа приложения
- `discord_bot/bot.py` - логика Discord бота
- `dashboard_server.py` - web-сервер панели управления  
- `database.py` - работа с базой данных
- `config.py` - загрузка конфигурации
- `setup.py` - мастер первоначальной настройки
- `index.html`, `styles.css`, `script.js` - web-интерфейс

## 🛠 Требуемые права бота

При добавлении бота на сервер выдайте ему права:
- Manage Messages (управление сообщениями)
- Kick Members (кик участников)
- Ban Members (бан участников)  
- Manage Channels (управление каналами)
- Manage Permissions (управление правами)
- Read Message History (чтение истории)
- Send Messages (отправка сообщений)
