# DiscordSer Bot

Мультифункциональный бот для Discord с красивой панелью управления и авто-setup.

## Что есть
- Discord-команды для модерации
- Тикеты
- Логи и варны
- Красивый сайт с анимациями и разделами управления
- Локальная админ-панель для настройки и модерации

## Запуск бота

```bash
pip install -r requirements.txt
python setup.py
python main.py
```

## Запуск панели

Запустите бота:

```bash
python main.py
```

В другом окне запустите панель:

```bash
python dashboard_server.py
```

По умолчанию панель открывается на `http://0.0.0.0:8000` и обычно публикуется через reverse proxy на домен вроде `https://zyc-discord.duckdns.org`.

Панель работает вместе с живым ботом и выполняет реальные действия на сервере.

## Запуск на домене

Если хочешь открыть панель по домену, используй reverse proxy (например, Nginx или Caddy) и направь домен `zyc-discord.duckdns.org` на порт `8000` локального Python-сервера.

Пример для Ubuntu:

- бот слушает локально `0.0.0.0:8000`
- Nginx принимает запросы на `zyc-discord.duckdns.org`
- Nginx проксирует их на `http://127.0.0.1:8000`

При желании можно изменить адрес/порт через `config.json`:

```json
{
  "dashboard_host": "0.0.0.0",
  "dashboard_port": 8000
}
```
